KT Merry reference

Exact KT Merry fonts found

KT Merry uses commercial fonts, not the Google Fonts in this preview page. The contact headline uses Silk Serif Extra Light, the form fields use Silk Serif Light Italic, and the small navigation text uses Mr Eaves Mod Regular. The logo itself appears as a wordmark image, so it should be treated as branding artwork rather than ordinary text.
